# ComfyUI-DigbyWan

This collection of nodes is intended to provide additional ways to use various Wan models. **These nodes are a WIP.  Future changes may break things.**

## Nodes

| Node Name | Description |
| --- | --- |
| [Wan 2.2 Middle Frame To Video](#wan-22-middle-frame-to-video) | Basic implementation of FMLF2V for Wan 2.2. 
| [Wan VACE 2.1 Video Smoother](#wan-vace-21-video-smoother) | Prepares VACE control_video to create a smooth transition between two videos, or within a single video at a specific frame.
| [Image Batch Loop Extractor](#image-batch-loop-extractor) | Extracts the middle frames of a video.  Use case is for a looping video that has been smoothed with itself.
| | |
